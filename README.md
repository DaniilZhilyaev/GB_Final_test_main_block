# Итоговая контрольная работа по основному блоку
## Задача:
Написать программу, которая из имеющегося массива строк формирует новый массив из строк, длина которых меньше, либо равна 3 символам. Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.
## Алгоритм решения:
1. Пользователь вводит в консоль через Enter элементы массива array
1. В массив threeSymbolsArray присваиваются элементы методом SelectArray от array:
	1. Все елементы массива array проверяются на условие длина меньше или равна 3 символам
	1. Удовлетворяющие условию значения записываются в строку elements через пробел
	1. С помощью раздлеления через пробел элементы записываются в новый массив res
	1. Метод возвращает массив res
1. В консоль выводится массив threeSymbolsArray
